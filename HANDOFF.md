## 📦 Что это за проект

Сайт подготовительного курса по информатике для иностранных
студентов (африканцы, поступающие в ПГУ, Россия). Основан на
учебнике ТГТУ 2018 г. Стек: PHP без фреймворка + Bootstrap 5 +
ванильный JS. Контент: 24 основных урока + 2 дополнительных +
интерактивные Pascal-тренажёры (уроки 19–24).

## 🗂 Структура (важное)

```
index.php             — главная
lessons.php           — обёртка для уроков, динамически грузит data/lessonN.php
extra-lessons.php     — список и просмотр доп. занятий
header.php            — общая шапка (используется index, extra-lessons)
footer.php            — общий футер
assets/
  css/style.css       — общие стили
  css/pascal-trainer.css  — стили только для уроков 19–24
  js/main.js          — общая интерактивность (TTS, drag&drop, проверки)
  js/pascal-trainer.js — интерпретатор Pascal + тренажёры
data/lesson1..24.php  — контент уроков (включается в lessons.php)
data/extra-lessons/   — контент доп. занятий
```

### assets/css/style.css — переписан
- Удалены **дубли**: блок `.l3-placeholder` шёл дважды, `.word-grouping .word-bank` определялся в двух местах.
- Добавлены стили `.lesson-grid` и `.lesson-grid__btn` (адаптивная сетка кнопок 1–24).
- Добавлено `.navbar-dark .nav-link.active` — подсветка активной вкладки (белая нижняя черта).
- Добавлено `.speak-btn.is-playing` — жёлтое состояние для играющей кнопки озвучки.
- На мобиле (`< 576px`) sticky-банк слов уменьшен с 35vh до 22vh — чтобы корзины оставались видны.

### assets/js/main.js — расширен
- Новая функция `initNavbarAutoClose()` — мобильное меню автоматически закрывается после клика по `.nav-link`.
- Функция `speak()` теперь принимает третий параметр `sourceBtn` и:
  - Меняет текст кнопки на ⏸ во время воспроизведения.
  - Добавляет/убирает класс `.is-playing` (жёлтый фон).
  - Возвращает оригинальный символ через `utterance.onend`/`onerror`.
  - Повторный клик по играющей кнопке ОСТАНАВЛИВАЕТ озвучку.
  - Сбрасывает состояние всех других играющих кнопок при старте новой.

### assets/js/pascal-trainer.js — добавлен touch
- В `initReorder()` добавлены обработчики `touchstart` / `touchmove` / `touchend` / `touchcancel`.
- При `touchmove` элемент перемещается между соседями списка через тот же `getDragAfter()`, что и десктопный drag.
- `e.preventDefault()` в touchmove только во время активного drag (не блокирует обычный скролл страницы).

### assets/css/pascal-trainer.css — мелкая правка
- К `.pas-reorder-item` добавлено `touch-action: none` — чтобы браузерный скролл не мешал нашему touch-обработчику.

1. **Откройте `/lessons.php?n=1`, `?n=10`, `?n=19`, `?n=24`** — все должны показываться и не ломаться.

5. **Озвучка**: нажмите ▶ на любой кнопке — она должна стать ⏸ + жёлтой. Дождитесь конца / нажмите ещё раз — должна вернуться в ▶.
